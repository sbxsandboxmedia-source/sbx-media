# SBX Media Website

Free GitHub Pages compatible static website.

## Setup
1. Upload `index.html`, `style.css`, and `script.js` to your GitHub repository.
2. Enable GitHub Pages from Settings > Pages.
3. Replace WhatsApp number `919999999999` with your number.
4. Replace Formspree action `https://formspree.io/f/YOUR_FORM_ID` with your real form endpoint.

## Free Form Options
- Formspree free plan
- Google Form link/button
- WhatsApp CTA
